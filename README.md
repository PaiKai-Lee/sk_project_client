sveltekit & Nestjs 練習專案。

功能參考公司的午餐系統，將錢預放在公司，進行線上扣款，不須每次都使用現金交易，並加入權限功能

## 當前功能

- 管理者創建 & 刪除使用者
- 使用者儲值 & 扣款
- 系統可以產生儲值/扣款紀錄，餘額確認

## 使用工具

- 前端使用 sveltekit(SPA)
- 後端使用 Nestjs with Prisma(Mysql)
- 部屬使用 Nginx

## 部屬

使用 Nginx 部屬(待補全)

1. 在您的伺服器上安裝 Nginx。
2. 在項目目錄中運行 npm run build 以構建前端，將 build 資料夾放置到 Nginx 預計指向位置。
3. 在項目目錄中運行 npm run start:prod 以啟用後端 API。
4. 配置 Nginx 以提供前端和後端。
